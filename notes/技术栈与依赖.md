# 技术栈与依赖

## 后端
- Java 17（`pom.xml`）
- Spring Boot 3.2.8
- Spring Web / Spring Data JDBC
- MyBatis 3.0.3、MyBatis-Plus 3.5.5
- Jakarta Persistence API 3.2.0
- MySQL Connector/J
- Hutool 5.8.25
- Lombok
- Spring Boot Starter Mail 3.3.0
- OpenAPI/Swagger：springdoc-openapi 2.5.0 + Knife4j 4.3.0
- 测试：spring-boot-starter-test、mybatis-spring-boot-starter-test

## 运行与配置要点
- 服务端口：`server.port=8888`
- 数据源：MySQL（`edums` 库）
- OpenAPI/Swagger：`/swagger-ui.html`

## 前端
