# 数据库表清单

## 表清单
- admin — 管理员信息表（字段：`数据库表字段-admin.md`）
- classes — 班级信息表（字段：`数据库表字段-classes.md`）
- course_open — 授课计划表（字段：`数据库表字段-course_open.md`）
- course_schedule — 课程安排表（字段：`数据库表字段-course_schedule.md`）
- courses — 课程信息表（字段：`数据库表字段-courses.md`）
- notice — 通知公告表（字段：`数据库表字段-notice.md`）
- setting — 教学配置表（字段：`数据库表字段-setting.md`）
- teachers — 教师信息表（字段：`数据库表字段-teachers.md`）
- teaching_incidents — 教学事故记录表（字段：`数据库表字段-teaching_incidents.md`）
